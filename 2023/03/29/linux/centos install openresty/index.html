centos install openresty

本文最后更新于：2023年9月1日 上午

centos install openresty

安装前准备：

你必须将这些库 perl 5.6.1+，libpare, libssl 安装到您的电脑里。对于linux来说，需要确认使用ldconfig命令，让其在你的系统环境路径中能找到它们。

centos 建议使用yum安装以下的开发库：

yum install pcre-devel openssl-devel gcc curl

构建OpenResty：

在centos 系统中添加openresty仓库，便于未来安装和更新软件包（通过yum check-update 命令）。运行下面的命令就可以添加我们的仓库。（对于Centos 8 以上版本，应将下面的yum都替换成dnf）：

wget https://openresty.org/package/centos/openresty.repo

sudo mv openresty.repo /etc/yum.repos.d/

sudo yum check-update

然后就可以像下面这样安装软件包，比如openresty:

sudo yum install -y openresty

如果你想安装命令行工具 resty，那么可以像下面这样安装 openresty-resty包：

sudo yum install -y openresty-resty

命令行工具 opm 在 openresty-opm包里，而restydoc 工具在 openresty-doc包里头。

直接运行/bin/openresty就可以启动OpenResty

/usr/local/openresty/bin/openresty

openresty

停止openresty

/usr/local/openresty/bin/openresty -s stop

openresty -s stop
IT
#openresty
centos install openresty
